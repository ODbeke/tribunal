# Tribunal Frontend Terminal

This directory contains the user interface for **Tribunal**, the Decentralized AI Escrow Court. It is built as a highly responsive Next.js application designed to interact with the GenLayer Bradbury Testnet.

## Technology Stack
* **Framework:** Next.js 15 (App Router, TypeScript)
* **Styling:** Tailwind CSS v4 & custom HSL obsidian design system tokens
* **Web3 Integration:** `genlayer-js` ( Bradbury chain RPC reading/writing)
* **Animations:** `framer-motion` for spring-based SVGs and micro-interactions
* **Icons:** `lucide-react`

## Quick Start
```bash
# Install dependencies
npm install

# Run the development environment
npm run dev

# Compile optimized static files
npm run build
```
